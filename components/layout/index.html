Layout 布局 ​

基础用法 ​

通过row和col组件，可以快速创建基础栅格布局，col组件的span属性可以帮助我们自由的进行布局组合

row共分为24列，你可以通过设置colspan属性，控制其占据多少列

基础布局使用
<template>
  <div class="sample">
    <qf-row>
      <qf-col>
        <div class="col"></div>
      </qf-col>
    </qf-row>

    <qf-row>
      <qf-col :span="12">
        <div class="col"></div>
      </qf-col>
    </qf-row>

    <qf-row>
      <qf-col :span="12">
        <div class="col"></div>
      </qf-col>
      <qf-col :span="12">
        <div class="col"></div>
      </qf-col>
    </qf-row>

    <qf-row>
      <qf-col :span="8">
        <div class="col"></div>
      </qf-col>
      <qf-col :span="8">
        <div class="col"></div>
      </qf-col>
      <qf-col :span="8">
        <div class="col"></div>
      </qf-col>
    </qf-row>
  </div>
</template>
<style scoped>
.sample {
  .qf-row {
    margin-bottom: 20px;
  }
  .col {
    width: 100%;
    height: 26px;
    border-radius: 4px;
    background-color: var(--primary-color);
  }
}
</style>

列偏移 Offset ​

col组件的offset属性可以控制列的偏移，满足不同场景需求

列偏移

      <qf-col :span="12" :offset="4">
        <div class="col"></div>
      </qf-col>

Row 字段 ​

参数说明类型可选值默认值
gutter栅格间隔number0
justify栅格水平排列方式enumstart/end/center/space-around/space-between/space-evenlystart
align栅格垂直布局enumtop/middle/bottom—

Col 字段 ​

参数说明类型可选值默认值
span栅格所占列数number24
offset栅格左侧的间隔列数numer0
